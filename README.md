# Centro Espírita Ogum Naruê

Site institucional desenvolvido com Flask e Jinja2.

## Como executar

1. Crie um ambiente virtual.
2. Instale as dependências:
   `pip install -r requirements.txt`
3. Execute:
   `python app.py`

## Rotas disponíveis

- `/`
- `/historia`
- `/agenda`
- `/nossa-casa`
- `/faq`
- `/contato`
